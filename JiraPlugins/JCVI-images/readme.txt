These files need to be installed in <JIRA_APP_DIR>/atlassian-jira/images for a backup of a JCVI JIRA to have the right banners.

For production (pepe) JIRA this is: /opt/software/jira/atlassian-jira/images
For development (pepe-dev) Sampletracking JIRA this is: /opt/www/sampletracking/jira/atlassian-jira/images
